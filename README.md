# synsdev/homebrew-syns

Homebrew tap for the [syns CLI](https://github.com/synsdev/syns-cli).

Auto-bumped on every `synsdev/syns-cli` release by cargo-dist's `publish-homebrew-formula` job.

## Usage

```bash
brew tap synsdev/syns
brew install syns
```

(Currently requires the upstream `synsdev/syns-cli` repository to be public for the `brew install` step to fetch the release tarball. The tap formula itself is public.)
